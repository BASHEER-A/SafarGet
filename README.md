# SafarGet

SafarGet is a native macOS download manager inspired by IDM.

## Features
- Multi-connection accelerated downloads
- Torrent support
- YouTube video downloads
- Native Apple Silicon support
- Safari & Chrome integration

## System Requirements
- macOS 13.5 or later
- Apple Silicon (M1/M2/M3)
- intel

This project is actively developed.
